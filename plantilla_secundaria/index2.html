Ejercicios básicos

Sigue las instrucciones para obtener la mayor puntuación.

Gato que camina

Utiliza solo los dos disfraces de el "gato" y haz que avance a la derecha cuando se presione la tecla "d".
Para esta tarea solo podrás utilizar 5 bloques
Pistas:Utiliza bloques como cambiar disfraz, esperar x tiempo, mover x pasos y al presionar x tecla.

Chutar un balón

Tienes que conseguir que cuando se presione la bandera verde, se reinicien las posiciones y
que cuando se presione la tecla "Q" un personaje cualquiera chute el balón.
(ÉL balón tiene que desplazarse y el personaje tener una animación).

Posición random

Este ejercicio consiste en conseguir que un personaje cualquiera se mueva a una posición aleatoria cada vez que el ratón se sitúe encima suya. Necesitarás:
bloques de posición y de variables.
